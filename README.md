# README

## Settings

- Clone repository
  git clone https://github.com/jacksteven7/hodinkee_blog.git

- Go inside the project
  `cd hodinkee_blog`

- Bundle the project
  `bundle install`

- Create databases
  `rake db:create`

- Run migrations
  `rake db:migrate`

- Install react dependencies
  `yarn install`

- Start server
  `rails s`

- Run specs

in another tab, run `rspec` to run all the specs
